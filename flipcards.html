Discover My Web Design Projects:


To explore the projects I've completed during my web design course, simply hover over each flip card to reveal more information about each assignment. You can click on the project titles to visit the respective project pages and experience my work firsthand. Enjoy exploring my web design journey!

Break the Ice!

This was our first project, and it was also the first thing I had ever uploaded to Github. For this project, I demonstrated how to use header, paragraph, and list tags. I also included an optional hyperlink.

Keeping it Stylish

This was our first assignment using CSS. I created three different boxes using different dimensions and colors. I also learned how to style and position header tags, and remove the web page border. I used internal styling for this project.

IU in the News

For this project, we designed a webpage inspired by IU news. This was the first project that used an external style sheet. It was the first time we used nav, article, aside, and footer tags. We also learned how to link images. I included a link for users to send me an email.

Holy Grail Layout

This assignment taught us how to use the holy grail layout and z-index. We also learned how to use the positioning property in CSS. To complete the project, I used fixed, relative, and absolute positioning. I applied my choice of colors and styling.

Pokemon Card

This was our first group assignment. We used the repl.it platform to collaborate. One member created a wireframe diagram and another started writing the basic HTML and CSS. I focused on finding the different images and applied the final styling and positioning for the completed project.

Centuries of Presidents

This was our first project that used multiple HTML pages. We used semantic markup, external styling, and hyperlinks for navigation. This project brought together the styling and positioning skills we practiced in the Pokemon card project. We also practiced using and resizing images.

Pinterest Pins

For this project, we created Pinterest style cards. This project required external, internal, and inline styling. I used my labrador, Ziggy, and my garden for inspiration. I learned how to superimpose titles over images. I focused on using colors from Adobe color wheel to match the card contents.

Indy's Best Eats

For this project, we created a prototype web suite for the Indianapolis Food Fair. We used a provided wireframe to create a landing page for different food categories. I then chose my own food categories and images for the other pages. I learned how to use an image as a hyperlink for this project.

Flexbox Coupons

This was our first project using flexbox and media queries. We used supplied images, basic HTML, and style specs. We then wrote the CSS needed to match the styling specs and media queries with two breakpoints. The result was ugly, as intended, but a great learning experience. Flexbox is definitely better than manual positioning!

Cyber Teams

This was one of my favorite projects, because the only requirements were to make three pages with a cybersecurity theme. I really appreciated the opportunity to make a project using my prefered styling and ideas. I made a landing page, a red team page, and a blue team page. I used flexbox to make the pages responsive.

Off the Grid!

For this project, I made a three page layout with an off grid living theme. We learned how to use the grid styling in CSS. I used grid for the layout and flexbox for some content. I used Adobe color wheel to find a color theme, and populated the content using a provided website for inspiration. Using grid, flexbox, and media queries made the pages responsive.

Signs of Spring

For this project, I created a spring themed page using flex cards. The webpage had six cards that used images from my garden in Spring. I used appropriate colors from Adobe color wheel, box shadowing, and hover effects for a great spring style. Clicking the cards takes you to sites where you can learn more about the plants in my garden. The flex cards and images were also made responsive to browser resizing.

Rise & Shine

This was a similar project to "Signs of Spring", but I chose a breakfast theme for this one. I used the color theme from CrackerBarrel. Then I searched for healthy breakfast items and images. I like all these breakfast items except shredded wheat. Yuck! Clicking the cards takes you to sites where you can learn more about the item. The flex cards and images were also made responsive to browser resizing.

Starbucks Homebrew

This was another group assignment where my group used repl.it to collaborate. One member tracked down recipes and images, while the other members worked on the HTML and CSS. We then combined our work for the finished project. We used the Starbucks color scheme and made our flipcards resize to make reading the recipes easier.

Dream Cars

For this project I was able to choose my own theme. I created a landing page with an informative article and an aside with links to my flipcard page and places to buy cars. I used vivid images to draw attention to the cars, and included info and links to my favorite cars. Like the other projects the page was responsive using flexbox and media queries.
